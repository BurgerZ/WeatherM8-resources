Weather Mate (M8)


Izin

RECEIVE BOOT COMPLETED - untuk memulai pembaruan latar belakang setelah reboot telepon
READ/WRITE EXTERNAL STORAGE - untuk membuat screenshot (fungsi berbagi)
INTERNET - perizinan internet (mengakses data dari internet)
ACCESS NETWORK STATE - periksa apakah internet dapat diakses
LOCATION - lokasi perangkat (meminta koordinat Anda untuk memuat perkiraan cuaca di lokasi Anda saat ini)
GET ACCOUNTS - Ini adalah apa yang akan Anda dapatkan sebagai peringatan perizinan "Kontak". Izin ini ditunjukkan sebagai Kontak, karena berada dalam grup perizinan Kontak (ini tidak akan membiarkan aplikasi membaca daftar kontak Anda). Ini memungkinkan aplikasi mendapatkan daftar terdaftar di email akun perangkat Anda untuk memeriksa apakah akun Anda ada dalam daftar bebas iklan.
