Permissões

ACESSO APÓS INICIALIZAÇÃO - Permite que uma actualização inicie depois do dispositivo ter reiniciado
LER/ESCREVER NO ARMAZENAMENTO EXTERNO - Permite o armazenamento de capturas de ecrã para utilizar a função partilhar
INTERNET - Acesso à Internet para obter dados
ACESSO AO ESTADO DA REDE - Verificar se o acesso à Internet está disponível
LOCALIZAÇÃO - Localizar dispositivo (obter as coordenadas do dispositivo para transferir a informação do tempo para o local actual
ACESSO A CONTAS - É um aviso para obter uma permissão de "Contactos". Esta permissão é exibida como Contactos porque se encontra no grupo de permissões de contactos (não vai permitir que a aplicação leia a sua lista de contactos). É permitido que a aplicação obtenha a lista de contas de email registadas no seu dispositivo para verificar se a sua conta se encontra na lista de sem anúncios
