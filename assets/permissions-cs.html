Počasí (Weather M8)


Oprávnění

RECEIVE BOOT COMPLETED - umožňuje aplikaci spuštění a aktivitu na pozadí po spuštění systému zařízení.
READ/WRITE EXTERNAL STORAGE - umožňuje aplikaci čtení obsahu USB a zapisovat do úložiště USB.
INTERNET - umožňuje aplikaci vytvářet síťové sokety a používat vlastní síťové protokoly pro připojení k internetu.
ACCESS NETWORK STATE - umožňuje aplikaci zjistit informace o síťových připojeních k internetu.
LOCATION - umožňuje aplikaci zjistit vaši aktuální polohu pro načtení aktuálních informací o počasí.
