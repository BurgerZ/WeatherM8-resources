Permissions

RECEIVE BOOT COMPLETED - 휴대폰을 재부팅한 후에 백그라운드 업데이트가 시작됩니다
READ/WRITE EXTERNAL STORAGE - 스크린샷을 찍습니다 (공유 기능)
INTERNET - 인터넷 권한 (인터넷에서 데이터 가져옴)
ACCESS NETWORK STATE - 인터넷이 접속 가능한지 확인
LOCATION - 기기의 위치를 확인함 (현재 위치의 예보를 불러오기 위해 좌표를 가져감)
GET ACCOUNTS - 이는 "연락처" 사용에 관한 경고입니다. 이 권한은 연락처 그룹에 있으므로 연락처로 표시됩니다 (앱이 연락처 목록을 읽도록 허용하지 않습니다). 사용자가 장치에 등록된 이메일을 가져와 광고 제거 목록에 당신의 계정이 있는지 확인할 수 있습니다..
