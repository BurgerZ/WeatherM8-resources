Погода M8


Разрешения

RECEIVE BOOT COMPLETED - запуск виджетов, уведомлений, а также фонового обновления прогноза погоды после перезагрузки устройства.
READ/WRITE EXTERNAL STORAGE - доступ к файлам устройства, необходим для функции "Поделиться" (для сохранения и чтения скриншота прогноза), а также для доступа к файлам устройства при прикреплении их к отчету ("Написать разработчику").
INTERNET - доступ к сети Интернет.
ACCESS NETWORK STATE - проверка состояния подключения к сети Интернет.
LOCATION - определение текущего местоположения вашего устройства.
