Надвор'е M8


Дазволы

RECEIVE BOOT COMPLETED - запуск віджэтаў, апавяшчэнняў, а таксама фонавага абнаўлення прагнозу надвор'я пасля перазагрузкі прылады.
READ/WRITE EXTERNAL STORAGE - доступ да файлаў прылады, неабходны для функцыі "Падзяліцца" (для захавання і чытання скрыншота прагнозу), а таксама для доступу да файлаў прылады пры прымацаванні іх да справаздачы ( "Напісаць распрацоўніку").
INTERNET - доступ да сеткі Інтэрнэт.
ACCESS NETWORK STATE - праверка стану падлучэння да сеткі Інтэрнэт.
LOCATION - ызначэнне бягучага месцазнаходжання вашай прылады.
